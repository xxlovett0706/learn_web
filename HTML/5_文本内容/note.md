# 文本内容

全是 block 块级元素

## 常用元素

### `<div>`

content division 内容划分元素

```html
<div></div>
```

### `<p>`

paragraph 段落元素

```html
<p>这是一个段落</p>
```

### `<ul>` 和 `<li>`

unordered list 无序列表元素

list item 列表项元素

```html
<ul>
    <li>JavaScript</li>
    <li>Rust</li>
    <li>MySQL</li>
</ul>
```

### `<ol>` 和 `<li>`

order list 有序列表元素

list item 列表项元素

```html
<ol>
    <li>娱乐</li>
    <li>学习</li>
    <li>工作</li>
</ol>
```

### `<dl>`、`<dt>`和`<dd>`

description list 描述列表

description term 描述术语

description detail 描述详情

```html
<dl>
    <dt>Firefox</dt>
    <dd>
        A free, open source, cross-platform, graphical web browser developed by
        the Mozilla Corporation and hundreds of volunteers.
    </dd>
</dl>
```
